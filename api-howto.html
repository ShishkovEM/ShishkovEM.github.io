Функционал отображения списка инвестидей Help

Описание экранных форм и работы с API отображения списка инвестидей

Диаграмма последовательностей, демонстрирующая процесс отображения списка инвестидей:

1. Экран "Подбор рекомендаций от аналитиков"

Описание

Этот экран отображает список инвестиционных идей. Пользователи могут просматривать различные инвестиционные идеи, включая название, тип, потенциальную доходность и дату обновления.

Работа с API

  • Запрос: GET /invest_ideas

  • Действие: Получение списка всех доступных инвестиционных идей.

  • Ответ API: Список инвестиционных идей с основной информацией.

2. Экран "Детальное описание инвестиционной идеи"

Описание

На этом экране отображается подробная информация об определенной инвестиционной идее, включая полное описание и ссылку на дополнительные материалы.

Работа с API

  • Запрос: GET /invest_ideas/

  • Действие: Получение детальной информации об инвестиционной идее по её идентификатору.

  • Ответ API: Полная информация об инвестиционной идее.

3. Экран "Фильтрация инвестиционных идей"

Описание

Этот экран позволяет пользователям фильтровать инвестиционные идеи по типу, дате обновления и потенциальной доходности.

Работа с API

  • Запрос: POST /invest_ideas/filter

  • Данные запроса: Типы ценных бумаг, критерий сортировки и направление сортировки.

  • Действие: Отображение отфильтрованного списка инвестиционных идей в соответствии с выбранными параметрами.

  • Ответ API: Отфильтрованный список инвестиционных идей.
